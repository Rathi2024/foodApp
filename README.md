# foodApp
<br>
Online Food Ordering System
<br>
Technologies Used: HTML, CSS, JavaScript, Java, JDBC, MySQL
<br>
Developed a food ordering web application with login and signup functionality.
Connected frontend pages with Java backend using Servlets and JDBC.
Designed and managed MySQL database for users, food items, and orders.
Implemented database connectivity using MySQL Connector and Apache Tomcat server.
Created responsive frontend pages using HTML and CSS.
